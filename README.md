vid2vid for Jetson AGX Xavier
======================================
Provide change to run vid2vid on Jetson platform
</br>
</br>

## Installation steps
**Flash JetPack4.1.1**
</br>
</br>
</br>

** Build pyTorch from source **
```C
$ ./install_pyTorch_Xavier.sh
```
</br>
</br>

** Install vid2vid on Jetson **
```C
$ ./install_vid2vid_Xavier.sh
```
